Problem Statement:
Students of VIT Bhopal frequently need transportation to nearby railway stations such as Bhopal Junction, Rani Kamlapati, and Sehore.
Currently, there is no unified platform where students can easily book a taxi or drivers can manage trip status.


Scope of the Project:
The scope of this project includes:
User (Student) Module :  Login using a predefined username and password
                         Book a taxi from VIT Bhopal to one of the available destinations
                         View all previously booked rides

Driver Module: Login using a driver account
               View unassigned trips
               Accept a booked trip
               Update trip status (Accepted / Completed)


Target Users:
This project is designed for:
 1. VIT Bhopal Students
Students who need quick transportation from the campus to nearby stations.
They can check routes, book taxis, and view their bookings.

2. Drivers Associated with Campus
Drivers who provide transportation services to VIT Bhopal students.
They can check available trips, accept bookings, and update trip status.


High-Level Features:
1. User Login System
Secure login using predefined credentials stored in lists.

2. Taxi Booking
Users can book taxis for:
VIT → Bhopal Junction
VIT → Rani Kamlapati
VIT → Sehore
Each booking includes:User
                      Route
                      Fare
                      Driver
                      Status

3. Driver Login System
Drivers can access their own dedicated dashboard.

4. Trip Management
Drivers can:
View unassigned trips
Accept a trip
Complete a trip


5. View Bookings
Students can view all their previous bookings along with updated trip status.
